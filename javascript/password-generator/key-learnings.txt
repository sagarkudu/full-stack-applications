1. DOM Manipulation

- learned the difference between .textContent and .value
- The .textContent works with elements like buttons, paragraphs, and divs.
- The .value works with form elements such as input fields.

e.g
button.textContent = "Generate Password";
input.value = "GeneratedPassword";
